## 制作について

このサイトは、HTMLとCSSを学びながら、さまざまな情報を調べて制作した作品です。  
デザインやレイアウト、画像の扱い方など、Web制作の基礎を一つ一つ確認しながら組み立てました。  
初めてのポートフォリオ作品として、ECサイトの基本的な構造を再現しています。
